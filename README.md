# QCameraControl

Simple Qt/QML application that displays all connected webcams streams.

The project uses QtMultimedia for webcam access.

## Pre-requisites

You need Qt 5, QMake, QML Quick and QtMultimedia.

## Build

```
qmake
make
```

## Usage

To run from the project directory:

`./qcameracontrol`
